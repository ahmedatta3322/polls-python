
## Installation


### clone the repo
```bash
  git clone [repository URL]
```
### run env
```bash
  python3 -m venv env
```
### Activate env
```bash
  source env/bin/activate
```
### install requirments
```bash
  pip install -r requirements.txt
```
### run the server
```bash
  python manage.py runserver
```
